# 🎵 Fresh Richie - Artist Portfolio

A modern, responsive portfolio website for the Mexican artist Fresh Richie. Built with React, TypeScript, and Tailwind CSS.

## ✨ Features

- **Responsive Design** - Works perfectly on all devices
- **Modern UI/UX** - Clean, professional design with smooth animations
- **Music Integration** - Spotify integration and YouTube videos
- **Real Content** - Authentic photos and information from Fresh Richie's social media
- **Performance Optimized** - Fast loading and smooth interactions

## 🎯 Sections

- **Hero** - Main landing section with artist photo
- **Music** - Discography with real tracks from Spotify
- **Videos** - YouTube integration with real videos
- **Shows** - Tour dates and concert information
- **News** - Latest updates and news about the artist
- **Store** - Merchandise and music sales
- **About** - Artist biography and information
- **Press Kit** - Media resources for press
- **Contact** - Contact information and social links

## 🚀 Tech Stack

- **React 19** - Modern React with hooks
- **TypeScript** - Type-safe development
- **Vite** - Fast build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **Responsive Design** - Mobile-first approach

## 🛠️ Development

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/fresh-richie-portfolio.git
cd fresh-richie-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

# 📸 Guía para Descargar Fotos de Instagram

## 🎯 Método Rápido y Fácil

### **Paso 1: Abrir Instagram**
1. Ve a: https://www.instagram.com/freshrichie.vhs/p/DN1Qn283E8l/
2. Abre cada foto en tamaño completo

### **Paso 2: Descargar Imágenes**
Para cada foto que quieras usar:

1. **Haz clic derecho** en la imagen
2. **Selecciona "Guardar imagen como..."**
3. **Guarda en la carpeta `images/`** con nombres descriptivos:
   - `hero_photo.jpg` - Para la foto principal
   - `artist_photo.jpg` - Para la foto del artista
   - `song_1.jpg` - Para "YA NO SOMOS NADA?"
   - `song_2.jpg` - Para "Luces de Neón"
   - `song_3.jpg` - Para "Ecos Rotos (EP)"
   - `song_4.jpg` - Para "Serendipia"
   - `song_5.jpg` - Para "Gravedad Cero"
   - `news_photo.jpg` - Para las noticias
   - `store_1.jpg` - Para música digital
   - `store_2.jpg` - Para vinilos
   - `store_3.jpg` - Para merchandising

### **Paso 3: Actualizar la App**
Una vez que tengas las fotos descargadas, ejecuta:

```bash
# Copiar las nuevas fotos a public/
copy "images\*" "public\"

# La app se actualizará automáticamente
```

## 🚀 Alternativa: Usar Herramientas Online

### **Opción 1: DownloadGram**
1. Ve a: https://downloadgram.com/
2. Pega el enlace de Instagram
3. Descarga las imágenes

### **Opción 2: InstaDownloader**
1. Ve a: https://instadownloader.co/
2. Pega el enlace del post
3. Descarga todas las fotos

## 📱 Usando el Navegador Móvil

Si estás en móvil:
1. Abre Instagram en el navegador
2. Mantén presionada la imagen
3. Selecciona "Descargar imagen"
4. Transfiere las fotos a tu computadora
